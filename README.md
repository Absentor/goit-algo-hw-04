## Task 1
Recursive file sorting by extension using:
- recursion
- pathlib
- shutil
- argparse

The program recursively scans directories, copies files, and sorts them into folders based on file extensions.

---

## Task 2
Visualization of Koch snowflake fractal using recursion and turtle graphics.

The user can specify the recursion level for generating the fractal.

---

## Task 3
Comparison of sorting algorithms:
- Insertion Sort
- Merge Sort
- Timsort (Python built-in sorted)

Execution time was measured using the `timeit` module.

---

## Conclusions

Insertion Sort works efficiently only for small datasets because its complexity is O(n²).

Merge Sort demonstrates significantly better performance on large datasets due to complexity O(n log n).

Timsort showed the best overall performance because it combines advantages of Merge Sort and Insertion Sort.

This demonstrates why built-in Python sorting functions are generally preferred in real-world applications.
